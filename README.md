# SaaS Product Engagement & Conversion Analysis

## Business Problem

A B2B SaaS company is experiencing below-benchmark trial-to-paid conversion and elevated churn among early-tenure subscribers. Leadership needs to understand which user behaviors during onboarding predict long-term retention, where users drop off in the activation funnel, and which customer segments deliver the highest lifetime value.

## Approach

**Methodology:** End-to-end analytics consulting workflow — Pyramid Principle / MECE problem structuring → Python EDA & feature engineering → SQL analytical queries → Tableau executive dashboard → McKinsey-style presentation deck.

**Tools:** Python (Pandas, NumPy, Matplotlib, Seaborn) · MySQL / DBeaver · Tableau Public · PowerPoint
